Project Overview

This high-impact project involved developing complete firmware and control software for a cutting-edge high-speed vision inspection and packing machine. The system achieves remarkable 0.7 second inspection cycle times while maintaining industrial-grade accuracy and reliability.

The solution integrates Hikrobot AI cameras with custom servo motor control and web-based HMI, creating a seamless end-to-end hardware and software pipeline that meets demanding industrial standards while maximizing throughput and performance.

Technical Implementation

Hikrobot AI Vision

Integrated advanced Hikrobot AI cameras with custom vision processing algorithms for real-time defect detection and quality control.

  • AI-powered defect detection
  • Real-time image processing
  • High-speed data acquisition

Custom Firmware

Developed specialized firmware for Leadshine L6 servo motors with advanced motion control profiles and real-time communication protocols.

  • Servo motor control
  • Real-time communication
  • Motion profiling

Communication Protocols

Implemented robust multi-protocol communication system including RS485 Modbus, Ethernet, and RS232 for reliable device integration.

  • RS485 Modbus integration
  • Ethernet communication
  • RS232 serial protocol

Web-based HMI

Created intuitive web-based Human Machine Interface using Flask for real-time monitoring, control, and system configuration.

  • Real-time dashboard
  • Remote monitoring
  • System configuration
