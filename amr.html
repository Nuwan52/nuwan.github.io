Project Overview

Designed and developed a small-scale Autonomous Mobile Robot (AMR) for factory automation at Phoenix Industries. The system integrates ROS2-based control, Gazebo simulation, and embedded hardware for real-time navigation and mapping.

Implemented SLAM (Simultaneous Localization and Mapping) for autonomous navigation, allowing the AMR to operate safely in factory environments. The project is being scaled using Nvidia Jetson and Nvidia Omniverse for large-scale training and simulation, targeting the factory’s first automated forklift system.

Technical Implementation

ROS2 Control

Developed robot control systems using ROS2, implementing publishers, subscribers, and real-time sensor integration.

  • Custom ROS2 nodes
  • Multi-sensor integration
  • Real-time feedback control

SLAM & Navigation

  • Mapping with LIDAR & sensors
  • Path planning & obstacle avoidance
  • Trajectory optimization

Embedded Hardware

Integrated Arduino and other embedded electronics for real-time control of motors, sensors, and actuators.
